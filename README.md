# Dirección de Servicios Académicos - UNSA

Sitio institucional de ejemplo en `Next.js` para la Dirección de Servicios Académicos de la Universidad Nacional de San Agustín de Arequipa.

## Incluye

- Página principal institucional y promocional
- Vista separada para `trámites/certificaciones`
- Diseño responsive
- Branding basado en los colores `#D1AA65`, `#83030A` y `#051F43`
- Integración visual de `SisacApp+` y logos UNSA

## Ejecutar localmente

```bash
npm install
npm run dev
```

Abrir `http://localhost:3000`.
